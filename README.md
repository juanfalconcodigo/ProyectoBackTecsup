# ***Backend Nodejs - Typescript - Socket.io***

### Es necesario tener instalado la última versión estable de [*Node*](https://nodejs.org/en/).

#### Reconstruir módulos de Node :
```
npm i
```
#### Levantar el servidor :
```
npx nodemon
```
#### Generar archivo para producción :
```
npm run build
```
